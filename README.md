# 📝 Task Manager Web App

A clean and responsive task management web application built using **HTML, CSS, and JavaScript**, designed to help users manage daily tasks efficiently.

---

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using **Local Storage**
- Clean and responsive user interface
- Real-time task updates

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage


## ⚙️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Start adding and managing tasks  

---

## 🎯 Purpose

This project was built to practice **frontend development fundamentals** and demonstrate the ability to build functional and user-friendly web applications without using external frameworks.
